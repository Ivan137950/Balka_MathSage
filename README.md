# Balka_MathSage

решение задачи изгиба многопродольной балки при помощи метода конечных разностей во фреймворке MathSage (Работает только под линукс/mac или можно запустить при в wsl). 
Задача: дана многопродольная балка, цель: построить эпюры деформации к ней аналитически (при помощи пакета math sage)  и численно (МКЭ на c++). При нулевой распределенной нагрузке решения в точности совпадают. Также в шарнирных опорах значения Q(x) , w(x) нулевые. Это означает, что представленное рещение задачи корректно.
